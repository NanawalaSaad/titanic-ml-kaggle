# titanic-ml-kaggle
Machine Learning Project Predicting Titanic Survival Using Logistic Regression,Random Forest, and XGBoost with feature Engineering and EDA.


🚢 Titanic Survival Prediction

This project uses Machine Learning models to predict passenger survival on the Titanic dataset.

📊 Models Used

Logistic Regression — 76.5% accuracy

Random Forest — 77.9% accuracy

XGBoost — 78.8% accuracy

📌 Key Steps

Data Cleaning (handling missing values)

Feature Engineering (FamilySize, Title extraction)

Exploratory Data Analysis (EDA)

Model Training & Evaluation

🏆 Result

Best Model: XGBoost

Kaggle Rank: 3,466
